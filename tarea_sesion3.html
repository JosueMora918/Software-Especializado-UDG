Links de HTML o Hiperenlaces

Los links se encuentran prácticamente en todos lados hablando de páginas web. Los links o ligas permiten a los usuarios navegar de una página a otra o de un sitio a otro. Los links se usan para abrir tanto páginas como recursos que pueden ser archivos o elementos multimedia, cuando colocas el puntero del mouse sobre un link, éste se convierte en una pequeña mano con un dedo apuntando hacia arriba. Los Links pueden ser de texto o incluso de imágenes

Imágenes de HTML

Las imágenes pueden dar un mejor diseño o aspecto a las páginas web. La etiqueta o tag que permite agregar imágenes a las pagínas web es la de img. Se sugiere usar imagenes de tipo JPG o GIF que no sean de gran tamaño para una más rápida descarga en Internet
